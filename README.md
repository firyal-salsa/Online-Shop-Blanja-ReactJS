# Blanja
  Blanja is an online shop web application. To find out information about shopping features (please click below), demo projects, executing programs, backend links, application screenshots, please see below.

## Features
<details>
  <summary>User</summary>
  
  ## Customer
  1. Sign up
  2. Login
  
  ## Seller
  1. Sign up
  2. Login
  3. Inventory
     * Create
     * Read
     * Read - Sold out
     * Update
     * Delete
</details>

<details>
  <summary>Category</summary>
  
  ## Category
  1. Read
</details>

<details>
  <summary>Product</summary>
  
  ## Product
  1. Create
  2. Read
  3. Update
  4. Delete
  5. Sort by name
  6. Sort by newest
  7. Sort by price
  8. Search
</details>

<details>
  <summary>Bag</summary>
  
  ## Bag
  1. Create
  2. Read
  3. Update
  4. Delete
</details>

## Executing program

* run the server
```
npm start
```
